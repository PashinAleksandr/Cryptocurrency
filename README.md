The Cryptocurrency Application

Features:
1) Downloading data from the server and updating the currency with a swipe to the bottom. 100 pieces.
2) Clicking on the currency opens the details screen.
3) Adding to favorites deleting from favorites and a separate Favorites screen
4) Downloading quote data from the server for: day, week, month, year and for all time
5) Changing the color of the price. The boost is red, the decrease is green.
6) Tracking Internet availability. In case of loss of connection, it displays a separate screen with a suggestion to reconnect when the Internet is available. Without the Internet, all actions with manets are blocked. It only works on a mobile device. It cannot be played in the xcode emulator due to a glitch in the Internet tracking library code.
7) It stores coin IDs and favorites in the device's long-term memory. Saves data after closing or restarting
8) Price update animation, 3 seconds after the new value, new color. After the time expires, it returns to the default color
